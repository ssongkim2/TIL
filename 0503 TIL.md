```
#include<stdio.h>

void SwapIntPtr(int **dp1, int **dp2)
{
	int *temp = *dp1;
	*dp1 = *dp2;
	*dp2 = temp;
	
}

int main(){
	int num1 = 10, num2 = 20;
	int *ptr1, *ptr2;
	ptr1=&num1, ptr2=&num2;
	printf("*ptr1, *ptr2: %d %d\n", *ptr1, *ptr2);
	
	SwapIntPtr(&ptr1, &ptr2);
	printf("*ptr1, *ptr2: %d %d", *ptr1, *ptr2);
	return 0;
}
```







더블포인터를 사용한 스왑함수!

**dp1 이 *ptr을 가르키고 

*dp1은 *ptr을 의미한다 따라서 스왑이 이루어 진다.
